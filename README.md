## 이웃집 대여소 Front-end
#### 사용 스택
* React, Apollo, Typescript
#### 주요 기능 특징
* 회원가입
  - twilio를 통한 휴대폰 번호 인증
* 로그인
  - jwt를 이용한 token 방식 인증
* 게시물 작성
  - 사진 업로드 구현
* 대여 신청 / 처리
